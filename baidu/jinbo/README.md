## 三、百度统计 / 爱番番代码位置

按爱番番后台说明，代码必须放在 `<head>` 标签之后、其它内容之前。两个页面均已正确放置：

```html
<head>
  <meta charset="UTF-8">

  <!-- 百度统计 / 爱番番 —— 这里是正确的位置 -->
  <script>
  var _hmt = _hmt || [];
  (function() {
    var hm = document.createElement("script");
    hm.src = "https://hm.baidu.com/hm.js?48925e86a0ee5f53acb94049f2d362b3";
    var s = document.getElementsByTagName("script")[0]; 
    s.parentNode.insertBefore(hm, s);
  })();
  </script>

  <title>...</title>
  ...
```

这个脚本加载后会自动：
1. 启动百度统计数据监测
2. 注入爱番番在线沟通弹窗（就是聊天入口）

所以页面上所有"找老师聊 / 立即咨询 / 在线咨询"的按钮都统一走 `openChat()`，由这个函数唤起爱番番弹窗（如果后台没开启弹窗，兜底会打开一个咨询链接，可自行替换）。

## 四、部署建议

1. **PC 页** → 部署到 `http://你的域名/bjjb/`（对应百度 PC 推广落地页）
2. **移动页** → 部署到 `http://你的域名/bjjb/m/`（对应百度移动推广落地页）
3. **UA 智能跳转**（可选）：在 Nginx / Apache 配置里判断 User-Agent，移动设备访问 `/bjjb/` 时自动 302 到 `/bjjb/m/`
